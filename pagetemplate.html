Foundation Theme information

The Hunter Foundation Theme is based on Foundation, a well known responsive front-end framework that many organizations use around the world. Hunter Industries websites are based on this framework.

Zurb's Foundation Documentation Foundation Templates

Examples

Foundation comes with a built in responsive framework. Foundation uses a "flexible 12 column layout" approach that offers a responsive behavior of its elements wrapping content.

Assign the parent div the "row" class and its children the "column" class; this will set up the responsive structure.

Foundation has three main specific breakpoints: small, medium and large. You have to append a number to each one of those classes that add up to 12. For instance, if you would like to split the layout for two elements at 50% on a large screen, assigning "large-6" classes to each one of them will make them behave in that manner. On the other hand if you want those two same elements to have a 60-40 proportion (e.g: sidebar) on a medium screen then you would have to assign them a "medium-8" and "medium-4" class respectively.
