App Standards

Buttons styles vary inside a browser app. Solid blue background colors are primarily used for actions that invove either a final step command or an action that will take the user somehere else in the app. For other actions, using the blue outlined buttons is recommend; they offer neutrality and don't stand out as much (preventing distraction).

Fontello

For Web Standards, Hunter uses a custom font family for corporate-defined icons using Fontello's front end framework. We have a defined set of icons used in our website that act as symbols inside a font family.
